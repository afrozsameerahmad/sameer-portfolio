# Sameer Portfolio

A modern personal portfolio website for **Sameer Ahmad** — an aspiring AI Engineer — showcasing profile, skills, experience, projects, certificates, blog, and contact information.

## 🌐 Live Preview

Deployed on Vercel:  
**https://sameer-portfolio-eight-orcin.vercel.app**

---

## ✨ Features

- Responsive and modern dark-themed UI
- Hero section with professional introduction
- Dedicated sections:
  - Home
  - About
  - Skills
  - Experience
  - Projects
  - Certificates
  - Blog
  - Contact
- Resume and contact call-to-action buttons
- Portfolio metrics (internships, projects, skills)
- Clean navigation and smooth visual hierarchy
- AI/Data Science-themed visual identity

## 🚀 Getting Started

### 1) Clone the repository

```bash
git clone https://github.com/afrozsameerahmad/sameer-portfolio.git
cd sameer-portfolio
```

### 2) Install dependencies

```bash
npm install
```

### 3) Run locally

```bash
npm run dev
```

Then open the local URL shown in your terminal (usually `http://localhost:5173` or similar).

---

## 🛠️ Build for Production

```bash
npm run build
```

To preview production build:

```bash
npm run preview
```
